📊 How to Check Results in Google Analytics

Real-time (Within Seconds):

  1. Go to analytics.google.com
  2. Reports → Realtime → Overview
  3. Click a test link above
  4. Watch it appear in real-time!

Full Reports (After 24 hours):

Option 1 - Pages Report (EASIEST):

  1. Reports → Engagement → Pages and screens
  2. Search for "murphy-redirect"
  3. You'll see TWO separate entries:
    • /murphy-redirect/advertiser - X views
    • /murphy-redirect/tribune - Y views

Option 2 - Events Report:

  1. Reports → Engagement → Events
  2. Find event: murphy_pdf_click
  3. Click on the event name
  4. Look for the "source" parameter to see breakdown

Compare Performance:

After the ads run, you'll see exactly which newspaper drove more traffic. Each source shows as its own "page" in GA4:
